jic

JIRA Command Line Client

View on GitHub

Is it so that you have to use JIRA, but hate the very idea of using WebUI? Got tired waiting for web page to load? Or your eyes got used to green-on-black world so much that JIRA WebUI's bright background is just painful for you?

Try jic - a command line client for JIRA, written in Python, lightweight and easy to install.

Currently development is done privately, but it is expected to go public by 25 November 2013. Stay tuned.

Authors and Contributors

This project is developed under the Linaro's umbrella - thanks to Linaro for helping engineers getting a more suitable interface to JIRA.

The project is led by @ototo, who is also the only developer so far, but your complaints, ideas and patches are welcome!
